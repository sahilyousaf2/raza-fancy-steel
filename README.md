# Raza Fancy Steel 🏗️

A premium, modern corporate website for **Raza Fancy Steel**, a leading industrial fabrication and steel works company based in Karachi, Pakistan. Built with a focus on performance, industrial aesthetics, and search engine optimization.

---

## 🚀 Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/) inspiration with custom premium glassmorphism.

---

## ✨ Key Features

- **Industrial UI/UX**: Ultra-premium dark theme with metallic textures and professional lighting effects.
- **Service Collections**: Interactive cards showcasing core capabilities like Warehouse Sheds, Solar Structures, and SS Works.
- **Project Portfolio**: A dynamic gallery of completed large-scale industrial projects across Pakistan.
- **Advanced Lead Generation**: 
  - Integrated contact forms with real-time validation.
  - Floating WhatsApp integration for instant inquiries.
- **Dynamic Animations**: Smooth scroll-triggered entrances and interactive hover states using Framer Motion.
- **Full Responsiveness**: Mobile-first design ensuring a seamless experience on all devices.

---

## 📈 SEO & Performance Optimizations

This project has been heavily optimized for search engines and core web vitals:

- **JSON-LD Structured Data**: Implemented `LocalBusiness` schema for enhanced local SEO in Karachi.
- **Dynamic Metadata**: Unique meta titles and descriptions for every page (Home, About, Services, Projects, Contact).
- **Image Optimization**: All images served via `next/image` with proper sizing, modern formats (WebP), and descriptive `alt` tags.
- **Core Web Vitals**: Optimized for Largest Contentful Paint (LCP) and Cumulative Layout Shift (CLS).
- **Search Engine Discovery**: 
  - Dynamic `sitemap.xml` generation.
  - Automated `robots.txt` configuration.
  - Canonical URL support.
- **Accessibility**: ARIA-compliant navigation and interactive elements for better screen reader support.

## 🛠️ Getting Started

### Prerequisites

- Node.js 18.x or later
- npm / yarn / pnpm

### Installation

1. **Clone the repository**:
   ```bash
   git clone [repository-url]
   cd raza-fancy-steel
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
